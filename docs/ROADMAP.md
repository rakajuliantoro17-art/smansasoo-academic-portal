# SMANSASOO Academic Portal Roadmap

Portal Pengumuman Akademik  
SMAN 1 Sooko Mojokerto

---

# Vision

Membangun portal akademik yang sederhana, cepat, aman, dan mudah dikelola menggunakan GitHub, Vercel, Google Apps Script, serta Google Spreadsheet sebagai pusat data.

Portal ini dirancang agar dapat digunakan setiap tahun tanpa perlu membangun ulang sistem.

---

# Technology Stack

- HTML5
- CSS3
- JavaScript ES6
- Google Spreadsheet
- Google Apps Script
- GitHub
- Vercel

# Development Roadmap

---

# Phase 1 — Foundation (Version 1.0)

Target:
Membuat portal yang berfungsi sebagai pengganti website lama dengan teknologi modern.

## Features

- Repository GitHub
- Deploy Vercel
- Responsive Website
- Google Spreadsheet Database
- Google Apps Script API
- Pencarian berdasarkan NIS
- Pencarian berdasarkan NISN
- Halaman hasil pengumuman
- Halaman error
- CSS Variables
- CSS Animation
- Service Worker
- Manifest
- Dokumentasi Project

Status:

✅ In Progress

# Success Indicators

## Version 1.0

- Website dapat diakses melalui Vercel.
- Data berhasil diambil dari Google Spreadsheet.
- Pencarian berdasarkan NIS/NISN berjalan.
- Tampilan responsif pada desktop dan mobile.

---

## Version 2.0

- Mendukung lebih dari satu jenis pengumuman.
- Konfigurasi dilakukan melalui Spreadsheet.
- Tidak perlu mengubah kode untuk pergantian tahun ajaran.

---

## Version 3.0

- Terintegrasi dengan layanan digital sekolah.
- Memiliki dashboard administrator.
- Mendukung statistik penggunaan.
